You can also select a specific child using querySelectorAll('.day')[3], or querySelector(':nth-child(4)'). Notice that Javascript indexes start at 0, which is why "Wednesday" is [3], vs CSS which counts elements from 1, which is why "Wednesday" is nth-child(4) with that method.
